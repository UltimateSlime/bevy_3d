# bevy_3d

Rust + Bevy 0.18 で開発中の 3D ゲーム。Skyrim 的なオープンワールド MMO を目指した学習プロジェクト。

## 環境

- Rust 1.94.1
- Bevy 0.18
- avian3d 0.5
- bevy_flycam 0.18
- OS: Windows 11 / GTX 1080 Ti

## 操作方法

| キー | アクション |
|---|---|
| WASD | 移動（カメラの向きに連動） |
| Space | ジャンプ（地面判定あり） |
| マウス | カメラ視点回転 |
| V | TPS / FPS 切り替え |
| Esc | 終了 |

## 現在の仕様

### カメラ
- **TPS モード**（デフォルト）: プレイヤーの後方上部からの三人称視点
  - カメラコリジョンあり（壁・地面へのめり込み防止）
  - Yaw/Pitch による自由な視点回転
- **FPS モード**: プレイヤーの目線視点

### プレイヤー
- カプセル型 Collider
- `RigidBody::Dynamic` + `LockedAxes::ROTATION_LOCKED`（転倒防止）
- レイキャストによる接地判定でジャンプ制御

### ワールド
- 地面: `Collider::cuboid` による静的コライダー
- 建物: 色・高さがランダムな箱型ビル群（静的コライダーあり）
- スカイボックス: キューブマップテクスチャ
